## 🚀 Como Finalizar no GitHub

### **Passo 1: Criar o Repositório no GitHub**
1. Acesse https://github.com/new
2. Nome: `Financeiro`
3. Descrição: `Sistema de Controle Financeiro - Aplicação web completa`
4. Privacidade: Public ou Private
5. **Não marque** "Initialize this repository with a README"
6. Clique em "Create repository"

### **Passo 2: Conectar e Fazer Push**
Execute um dos métodos abaixo:

#### **Método 1: Script Automatizado**
```bash
chmod +x setup_github.sh
./setup_github.sh
```

#### **Método 2: Comandos Manuais**
```bash
# Configure seu username do GitHub
git remote set-url origin https://github.com/SEU-USUARIO/Financeiro.git

# Faça o push
git branch -M main
git push -u origin main
```
